ServeRAID MR 10 k SAS/SATA Controller - Linux Installation Client and Basic Commands

Posted by kairo on Mon 12 January 2009

This post show installation of MegaCli (ServeRAID MR 10 k SAS/SATA Controller client) and two basic commands

1. Installation

Get ServeRAID CD and extract the file MegaCli-X.XX.XX-Y.i386.rpm.

Install using the rpm command

` # rpm -ivh rpms/MegaCli-3.00.07-1.i386.rpm Preparing...               

[100%] 1:MegaCli               

##################################### [100%]`

Listing the installed files with rpm command

# rpm -ql MegaCli /opt/MegaRAID/MegaCli/MegaCli /opt/MegaRAID/MegaCli/MegaCli64
