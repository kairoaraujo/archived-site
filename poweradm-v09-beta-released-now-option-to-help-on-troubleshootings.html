PowerAdm v0.9-beta released. Now option to help on troubleshootings

Posted by Kairo Araujo on Thu 09 April 2015

I'm releasing the version 0.9-beta of PowerAdm http://www.poweradm.org

Principal features:

  • troubleshooting options. I'm using a lsseas* and I create a similar tool to NPIV (inspired on the lsseas);

  • working with HMC 8;

  • improve on documantation based on pydoc.

* lsseas is an original code of Benoit Creau from http://chmod666.org. I modified the code to work with PowerAdm. Thanks Benoit Creau for the idea and original code.

About troubleshooting options

All screenshots are here https://github.com/kairoaraujo/PowerAdm/wiki/troubleshooting.
