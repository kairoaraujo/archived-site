Linux and AIX Users Locked (Usuários com status lock)

Posted by kairo on Wed 17 December 2008

Como identificar usuários com senhas em status lock

LINUX

Verificando status do usuário
# passwd -S kairo kairo LK 2008-10-31 0 99999 7 -1 (Password locked.)

Retirando lock do usuário

# passwd -u kairo Unlocking password for user kairo. passwd: Success.

o status fica normal

# passwd -S kairo kairo PS 2008-10-31 0 99999 7 -1 (Password set, MD5 crypt.)

Fazendo lock de senha do usuário

# passwd -l kairo Locking password for user kairo. passwd: Success

AIX
