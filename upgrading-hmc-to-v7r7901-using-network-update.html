Upgrading HMC to V7R7.9.0.1 Using network update

Posted by Kairo Araujo on Thu 05 March 2015

[sorry my bad english]

0. Check the firmware x HMC version matrix

http://www-933.ibm.com/support/fixcentral/main/transform?xml=http://download.boulder.ibm.com/ibmdl/pub/software/server/firmware/sfw_fixSupportedCombos.xml&title=Supported+Combinations

Don't update to HMC 7.7.90 if you have these firmwares

  • 01Ax770 levels 032 (GA) to 064

  • 01Ax773 levels 033 to 051

  • 01Ax780 levels 040 (GA) to 050

Check it for more details: http://www-01.ibm.com/support/docview.wss?uid=nas8N1020088

1. Download the V7R7.9.0

The files:

ftp> cd software/server/hmc/network/v7790

ftp> prompt

ftp> bin  

ftp> mget * 

ftp> bye  

2. On IBM Fix Central you need get files to upgrade to V7R7.9.0.1

http://www.ibm.com/support/fixcentral/main/quickorder?parent=powersysmgmntcouncil&product=ibm/hmc/9100HMC&release=V7R7.9.0&platform=All&function=all&source=fc

The required files is *.iso:

HMC_Update_V7R790_SP1.iso

MH01406.iso

MH01435.iso

MH01463.iso

MH01467.iso

MH01473.iso

MH01484.iso

MH01490.iso

Tip: Put this files in local FTP  too.

3. Upgrading to v7790

Put all files downloaded on item 1 in a local FTP Server

4. Login at HMC using SSH and run the commands (hscroot)

# saveupgdata -r disk

# getupgfiles -h ftpserver -u root --passwd xxxxx -d /tmp/v7790

# chhmc -c altdiskboot -s enable --mode upgrade

# hmcshutdown -r -t now

The upgrade time for my HMC was 30 minutes.

5. Updating to v7r7.9.0.1

Log on HMC Web interface

a. Updates (left menu)

b. HMC Update

b.1. Next

b.2. Remote Server

b.3. FTP , Remote Server Address, User ID, Password and Dir files

b.4. Select File

At this point, apply MHXXXX.iso files (one by one) using number sequence. If you receive the message requiring the version V7R79.0.1 is the momment to apply the HMC_Update_V7R790_SP1.iso . I don't know if this is correct method, but works fine to me.

All the .iso requires HMC reboot.

When you apply the  HMC_Update_V7R790_SP1.iso you will receive that screen (sorry, my HMC in Portuguese)

The reboot for Update is ~30 minutes
